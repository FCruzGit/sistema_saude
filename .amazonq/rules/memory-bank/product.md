# Product Overview

## Purpose
Sistema SUS - Farmácia Popular is a desktop application for managing popular pharmacies within Brazil's Unified Health System (SUS). It enables citizens to consult and acquire medications from Basic Health Units (UBS) while providing administrators with comprehensive management tools.

## Value Proposition
- Democratizes access to medications through the public health system
- Streamlines medication distribution at UBS facilities
- Provides transparent pricing and availability information
- Ensures controlled medication tracking through prescription validation
- Simplifies administrative tasks for health unit management

## Key Features

### For Citizens (Clients)
- **Account Management**: Secure registration with CPF validation, password recovery via security questions, and automatic account lockout after failed login attempts
- **UBS Discovery**: Browse and search health units by name or address with detailed location information and images
- **Medication Catalog**: View available medications with comprehensive details (description, dosage, type, pricing, stock levels, prescription requirements)
- **Shopping Cart**: Add multiple medications, adjust quantities, and review total costs before checkout
- **Order Processing**: Submit orders with automatic prescription upload for controlled medications, balance validation, and stock verification
- **Prescription Management**: Upload medical prescriptions in PDF, JPG, JPEG, or PNG formats for controlled medications

### For Administrators
- **UBS Management**: Create, update, and remove health units with complete address and image management
- **Medication Inventory**: Add, edit, and delete medications with pricing, stock levels, prescription requirements, and UBS assignment
- **User Administration**: Add funds to user accounts and unlock blocked accounts
- **Order Review**: Approve or reject orders requiring prescription validation with prescription viewing capability
- **System Monitoring**: View all orders, blocked users, and inventory levels across the system

## Target Users

### Primary Users
- **Citizens**: Individuals seeking affordable medications through the SUS program
- **Health Unit Staff**: Administrators managing UBS operations and medication distribution

### Use Cases
1. **Medication Acquisition**: Citizens browse available medications at nearby UBS facilities and place orders
2. **Prescription Validation**: Controlled medications require prescription upload and administrative approval
3. **Inventory Management**: Administrators maintain accurate medication stock levels across multiple UBS locations
4. **User Support**: Administrators assist users with account issues and balance management
5. **Order Fulfillment**: Staff review and approve orders, ensuring compliance with prescription requirements

## System Characteristics
- Desktop application with intuitive graphical interface
- Real-time data database using JSON storage
- Automated validation for security and compliance
- Visual design aligned with SUS branding (blue color scheme)
- Support for multiple concurrent users (separate instances)
